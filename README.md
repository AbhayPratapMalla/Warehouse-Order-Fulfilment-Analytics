# Warehouse & Order Fulfilment Analytics

## Project Overview

This project analyzes warehouse and order fulfilment performance using Microsoft Excel. The objective was to evaluate delivery efficiency, return patterns, warehouse operations, and revenue performance to identify operational bottlenecks and support data-driven decision-making.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboard Development
- KPI Analysis

## Key Business Questions

- How effective is the order fulfilment process?
- What factors contribute to product returns?
- Which product categories generate the highest revenue?
- Are delivery delays caused by warehouse operations or downstream logistics?
- How can fulfilment performance be improved?

## Key Findings

- Only 45% of total orders were successfully delivered.
- The return rate was 25.8%, primarily driven by damaged products.
- Average delivery time was approximately 7 days.
- Warehouse processing times remained relatively stable at 1.8–2 days.
- Clothing was the highest revenue-generating category, contributing approximately £1.53M.

## Recommendations

- Improve packaging and handling processes to reduce damage-related returns.
- Optimize delivery routing and logistics planning to reduce delivery delays.
- Prioritize inventory allocation for high-performing product categories.
- Standardize warehouse processes to improve operational consistency.
